##**To use Locations end points:**

**go run locationRest.go**

#**Sample POST request:**

http://localhost:8080/locations/

Body

{

   "name" : "John Smith",

   "address" : "123 Main St",

   "city" : "San Francisco",

   "state" : "CA",

   "zip" : "945113"

}

POST Response:

{
  "id": 98081,
  "name": "John Smith",
  "address": "123 Main St",
  "city": "San Francisco",
  "state": "CA",
  "zip": "945113",
  "coordinate": {
    "lat": 37.7917618,
    "lng": -122.3943405
  }
}

#**Sample GET Request:**

http://localhost:8080/locations/98081

GET Response

{
  "id": 98081,
  "name": "John Smith",
  "address": "123 Main St",
  "city": "San Francisco",
  "state": "CA",
  "zip": "945113",
  "coordinate": {
    "lat": 37.7917618,
    "lng": -122.3943405
  }
}

#**Sample PUT Request:**
